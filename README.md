# Portfolio Project

Ce projet a été réalisé dans le cadre de ma formation afin de créer un portfolio personnel mettant en avant mes compétences et mes projets. Il s'agit d'une application web développée avec le framework **Vue.js** et le bundler **Vite**, offrant une interface moderne et responsive.

##  Technologies utilisées

- **Vue.js** : Framework JavaScript progressif pour construire des interfaces utilisateur.
- **Vite** : Outil de build rapide pour les projets front-end.
- **CSS** : Pour le style et la mise en page.
- **JavaScript** : Pour les interactions dynamiques.
- **HTML** : Structure de la page web.

##  Installation et utilisation

1. **Cloner le dépôt :**

   ```bash
   git clone https://github.com/Dada-nol/Portfolio-project.git
   cd Portfolio-project
2. **Installer les dépendances :**

   ```bash
   npm install

3. **Lancer le serveur de développement :**

   ```bash
   npm run dev

##  Structure du projet

- `index.html` : Page principale du portfolio.
- `src/` : Contient les composants Vue.js et les fichiers JavaScript.
- `public/` : Fichiers statiques accessibles directement.
- `package.json` : Fichier de configuration du projet avec les dépendances.
- `vite.config.js` : Configuration de Vite.

##  Objectifs du projet

- Mettre en pratique les compétences acquises en **Vue.js** et en développement front-end.
- Créer une interface utilisateur responsive et attrayante.
- Présenter mes projets et compétences de manière professionnelle.
